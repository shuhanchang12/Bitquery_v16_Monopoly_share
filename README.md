**The Next-Gen AI Crypto Co-Pilot**

## ‚ú® Key Features

### 1. üìä Intelligent Dashboard
- **Real-Time Market Data**: Interactive charts and watchlist for top cryptocurrencies (BTC, ETH, SOL, XRP).
- **AI Market Analysis**: On-demand market summaries generated by Gemini 2.5 Flash.
- **Sentiment Cloud**: Visualizes trending narratives from X (Twitter).
  - **Interactive Nodes**: Click keywords to search resources.
  - **Entity Graph**: Expand topics into a mind map tracking related Wallets, Influencers, and Tokens.
  - **Copy Trading Simulation**: Simulate copying strategies from high-signal entities.
- **Algorithmic Trading Simulator**: Backtest strategies (e.g., Momentum Scalp) and receive an **AI Post-Trade Reflection Report** analyzing performance.

### 2. üí¨ AI Consultant (Chat)
- **Context-Aware Chat**: Discuss market trends, specific coins, or technical concepts.
- **Search Grounding**: AI responses are grounded in real-time Google Search results for accuracy.
- **Source Citations**: Direct links to sources used by the AI.

### 3. üéì Bitquery Academy (Education)
- **AI-Powered Modules**: Learn concepts like Blockchain Basics, DeFi, and Technical Analysis.
- **NotebookLM Mode**: 
  - **Audio Overviews**: Generates a conversational "podcast" script (Host A & Host B) explaining the topic.
  - **Key Takeaways**: Auto-generated study notes and summaries.

## üöÄ Getting Started

### Prerequisites

- Node.js (v18+)
- A Google AI Studio API Key

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/bitquery.git
   cd bitquery
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure API Key**
   Create a `.env` file in the root directory and add your Gemini API key:
   ```env
   API_KEY=your_google_genai_api_key_here
   ```
   > **Note**: The application expects `process.env.API_KEY` to be available.

4. **Run the development server**
   ```bash
   npm run dev
   ```

## üìñ Usage Guide

1. **Dashboard**: 
   - Click **"Refresh AI Data"** to fetch the latest market sentiment.
   - Click on any word in the **Sentiment Cloud** to open the **Intelligence Mind Map**.
   - Explore the **Entity Graph** to see connections between topics and wallets.
   - Run the **Algorithmic Trading Simulator**, stop it, and click "Run AI Post-Mortem" to get a strategy audit.

2. **Chat**: 
   - Ask questions like "Why is Solana up today?" or "Explain staking risks".
   - View sources linked in the chat bubbles for verification.

3. **Academy**: 
   - Select a module (e.g., "DeFi Explained").
   - Switch to **"NotebookLM Mode"** to read a podcast-style script and see key takeaways.
